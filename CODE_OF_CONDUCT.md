# Código de Conduta — Iris AI

Este projeto adota o Código de Conduta do Contributor Covenant para promover uma
comunidade saudável e acolhedora. Espera-se que todos os contribuidores sigam estas
regras básicas de convivência.

Resumo:
- Seja respeitoso e profissional.
- Não tolere assédio, linguagem discriminatória ou ataques pessoais.
- Respeite a privacidade de terceiros e não compartilhe dados sensíveis.

Leia a versão completa em: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Reportar violações: abra uma issue privada (ou contate o mantenedor `@mateusfavero`).
